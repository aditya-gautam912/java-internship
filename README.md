## 📌 Tasks Completed

### 🔢 Task 1: Simple Calculator (Console)
**Description:**
A basic calculator that performs:
- Addition
- Subtraction
- Multiplication
- Division

**Concepts Used:**
- Scanner (Input)
- Switch-case / If-else
- Basic arithmetic operations

---

### 🎓 Task 2: Student Grade Management System
**Description:**
A program to manage student grades:
- Input student name and marks (3 subjects)
- Calculate average
- Assign grades (A/B/C/Fail)
- Display student report

**Concepts Used:**
- Classes and Objects
- Arrays
- Conditional logic

---

### 🏧 Task 3: ATM Simulation
**Description:**
A menu-driven ATM system with:
- Check balance
- Deposit money
- Withdraw money
- Exit option

**Concepts Used:**
- Loops
- Conditional statements
- Methods
- Menu-driven programs

---

### 🎯 Task 4: Number Guessing Game
**Description:**
A simple game where:
- Random number (1–100) is generated
- User guesses the number
- Hints: "Too High" / "Too Low"
- Tracks number of attempts

**Concepts Used:**
- Random class
- Loops
- Conditional statements
